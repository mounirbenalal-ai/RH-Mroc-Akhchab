# Akhchab RH (المغربية للأخشاب) 🇲🇦🏗️

## 🌲 نظرة عامة عن المشروع (Overview)
**Akhchab RH** هو نظام إدارة الموارد البشرية (RH) المتكامل والمخصص حصرياً للشركة المغربية للأخشاب. تم تصميم هذا التطبيق ليعكس الهوية المغربية العريقة في صناعة الأخشاب، مع دمج حلول تكنولوجية حديثة لضمان الكفاءة، الأمان، والاستدامة.

هدفنا هو زراعة "روح" تكنولوجية خضراء في عمليات الإدارة، بما يتماشى مع معايير المسؤولية البيئية والاجتماعية (CSR).

## 🚀 المميزات الرئيسية (Features)
التطبيق مقسم إلى ثلاث واجهات رئيسية لضمان الخصوصية والفعالية:

### 👤 1. واجهة الموظف (Employee Interface)
* **تسجيل الحضور عبر GPS:** التحقق من وجود الموظف داخل حدود المصنع عند تسجيل الحضور والإنصراف.
* **الخدمات الذاتية:** طلب إجازات، عرض كشوف الرواتب، وتحديث البيانات الشخصية.
* **البلاغات البيئية:** الإبلاغ عن مخاطر بيئية أو أعطال في آلات الخشب.

### 🛡️ 2. واجهة الأمن والحراسة (Security Interface)
* **إدارة الدوريات:** تسجيل نقاط التفتيش داخل المصنع عبر رموز QR.
* **سجل الزوار والشاحنات:** تسجيل بيانات السائق ورقم اللوحة، والتقاط صورة للحمولة (أخشاب/نشارة).
* **البلاغات الطارئة:** زر أحمر بارز لإرسال بلاغ فوري عن حالة طارئة.

### 📊 3. لوحة تحكم الإدارة (HR Admin Dashboard)
* **إدارة الموظفين:** جدول شامل للموظفين مع بياناتهم وعقودهم.
* **الحضور والرواتب:** تقارير الحضور وتصديرها لحساب الرواتب.
* **لوحة معلومات الاستدامة:** رسوم بيانية لاستهلاك الخشب شهرياً ونسبة تدوير المخلفات.
* **سجلات الأمن:** جدول لعرض بلاغات الأمن، سجل الزوار، وسجلات الدوريات.

## 🎨 الهوية البصرية (Branding)
تم استيحاء الهوية البصرية للتطبيق من شعار الشركة ومن ألوان الغابات المغربية:
* **الأخضر الغابوي (#2D5A27):** يمثل الخشب الحي، الاستدامة، والهدوء.
* **الحليبي (Off-white):** يمثل لون الخشب الخام ونظافة الورق.

## 🛠️ المكدس التقني (Tech Stack)
* **الواجهة الأمامية (Frontend):** React (TypeScript), Tailwind CSS.
* **الواجهة الخلفية وقاعدة البيانات (Backend & DB):** Supabase (Postgres with RLS).
* **التحقق (Auth):** Supabase Auth.
* **الموقع الجغرافي (Geo):** Supabase Point & JavaScript Geolocation API.

## 📜 الأساس القانوني والخصوصية (Legal & Privacy)
* **سياسة الخصوصية:** يتم جمع بيانات GPS فقط عند تسجيل الحضور والإنصراف داخل حدود المصنع.
* **قانون الاشتراك:** التطبيق مخصص حصرياً لموظفي الشركة المغربية للأخشاب.
